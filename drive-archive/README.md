# Архив для Google Drive

Сюда кладите файлы, которые нужны в облаке, но не обязательны в git:

- исходники макетов, договоры, прайсы (PDF, DOCX);
- крупные фото до сжатия;
- скриншоты, инструкции для сотрудников.

После добавления файлов запустите:

```powershell
.\scripts\sync-to-google-drive.ps1
```

Рабочая разработка сайта — в локальной папке **вне** Google Drive (см. `CONTEXT.md`, раздел «Google Drive и Cursor»).
