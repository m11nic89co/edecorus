# edecorus — локальная копия

Репозиторий: [github.com/m11nic89co/edecorus](https://github.com/m11nic89co/edecorus)

## Быстрое клонирование

### Вариант 1: Git (рекомендуется)

Если Git установлен:

```powershell
git clone https://github.com/m11nic89co/edecorus .
```

Если Git не установлен:

```powershell
winget install Git.Git
```

После установки перезапустите терминал и выполните `git clone` снова.

### Вариант 2: Скрипт PowerShell

```powershell
.\clone.ps1
```

### Вариант 3: Ручная загрузка

1. Откройте в браузере: https://github.com/m11nic89co/edecorus/archive/refs/heads/main.zip  
2. Скачайте архив  
3. Распакуйте ZIP  
4. Скопируйте содержимое папки `edecorus-main` в эту директорию
